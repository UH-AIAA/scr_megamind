◆ calculateState()

void FLIGHT::calculateState ( )

gets flight state

The function uses a cascading switch case to determine which stage of flight the rocket is in. At each stage, it calls a helper function to determine if it should move to the next one.

◆ incrementTime()

void FLIGHT::incrementTime ( )

tracks time during flight

This function calls the millis() function from the Arduino library to track milliseconds since the board was powered on. It updates three variables:

  1. deltaTime_ms
  2. runningTime_ms
  3. output.totalTime_ms
